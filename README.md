# 로또 (Kotlin + Spring)

이번 오픈 미션 과제를 어떻게 할까 고민을 하던 중, 평소에 기회가 있으면 공부를 하고 싶었던 코틀린과 스프링을 이용해서 오픈 미션을 진행 해야겠다고 생각했습니다.  
코틀린은 처음이고, 스프링은 공부 겸 몇 번 사용해 본적은 있지만 아직 익숙하지 않기 때문에, 오픈미션을 하기에 괜찮지 않을까 생각했습니다.

구현할 언어를 정했으니, 그럼 무엇을 구현하는 게 좋을까 고민해 봤을 때, 기존에 진행했던 로또를 그대로 코틀린과 스프링으로 구현하는 것보다 조금 수정을 해서 실제 로또를 구매하는 시뮬레이터처럼 구현하는 것이 좋을 것 같아 이렇게 오픈 미션을 진행하게 되었습니다.

---

## 기능

1. 구매할 로또 금액을 입력
2. 수동으로 구매할 로또 개수 입력
   - 0~총 구매 개수 사이 숫자 입력
3. 수동으로 선택한 로또 번호 입력
4. 수동을 제외한 나머지 로또 번호는 자동으로 생성
5. 구매한 로또 번호 출력
6. 당첨 번호 6개와 보너스 번호 1개 자동 생성 후 출력
7. 구매한 로또 번호와 당첨 번호를 비교하여 일치 개수에 따라 등수 계산
8. 총 수익률 계산 및 출력

---

## 사용 방법

1. 구입 금액 입력
2. 총 구매할 로또 개수 입력
3. 수동으로 몇 개를 구매할지 입력
4. 수동 로또 번호 입력
